# 文件权限类型
Linux文件权限分为三种基本类型：
- **读（Read，r）**：允许查看文件内容
- **写（Write，w）**：允许修改文件内容，以及删除和重命名文件
- **执行（Execute，x）**：对于文件，允许作为程序执行；对于目录，允许进入目录并访问其内容

# 权限表示方法
在Linux中，文件权限以`-`、`r`、`w`、`x`来表示，通过`ls -l`查看。一个文件权限设置看起来像这样：
```
-rw-r--r--
```
每三个字符为一组，分别代表：
1. 第一组（`-rw-`）：文件所有者的权限，这里表示有读（`r`）和写（`w`）权限，但没有执行（`-`）权限
2. 第二组（`r--`）：文件所属组的权限，这里只有读权限
3. 第三组（`r--`）：其他用户的权限，也只有读权限

# 修改文件权限
可以使用`chmod`（change mode）来修改文件权限。`chmod`有两种方式来设置权限：
1. **符号表示法**：`chmod u+x file`表示给文件所有者添加执行权限
2. **八进制表示法**：每个权限位可以转换为一个八进制数（读4，写2，执行1），`chmod 755 file`表示设置所有者有读、写、执行权限（7），组和其他用户有读和执行权限（5）

# 实例
假设有一个名为`script.sh`的脚本文件，你想让所有用户都能执行它，但只有你能修改它：
1. 符号表示法：`chmod a+x script.sh`（给所有用户添加执行权限）。
2. 八进制表示法：`chmod 755 script.sh`（所有者有读写执行权限，组和其他用户有读执行权限）
